# Student Management API

Spring Boot REST API for managing student records.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- ModelMapper

## Features
- Create Student
- Get Student by ID
- Get All Students
- Update Student
- Delete Student

## API Endpoints

POST /students
GET /students
GET /students/{id}
PUT /students/{id}
DELETE /students/{id}
